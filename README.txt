AboManager - minimalistische Web App

Dateien einfach in dein GitHub-Repo oder auf Cloudflare Pages / Netlify hochladen.

Wichtig für GitHub Pages:
- Repository Settings -> Pages
- Source: Deploy from branch
- Branch: main
- Folder: /root

Die App speichert alle Abos lokal im Browser via localStorage.
Benachrichtigungen funktionieren nach Freigabe im Browser.
Für echte Background-Pushes auf allen Geräten bräuchtest du später ein Backend wie Firebase.
